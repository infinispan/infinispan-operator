= Changing log levels
Change levels for different {brandname} logging categories.

.Procedure

. Configure {brandname} logging with the `spec.logging.categories` field in your `Infinispan` CR.
+
[source,options="nowrap",subs=attributes+]
----
include::yaml/logging_categories.yaml[]
----
+
[NOTE]
====
The root logging category is `org.infinispan` and is `INFO` by default.
====
+
. Apply the changes.
. Retrieve logs from {brandname} nodes as required.
+
[source,options="nowrap",subs=attributes+]
----
$ {oc_logs} -f $POD_NAME
----
