= Configuring pod termination grace period

[role="_abstract"]

Optionally configure the `TerminationGracePeriodSeconds` of the {brandname} pods.

The {ispn_operator} relies on the Kubernetes default `TerminationGracePeriodSeconds` unless a value is explicitly provided.
We only recommend providing your own value once you have determined that the default value do not match your requirements.
